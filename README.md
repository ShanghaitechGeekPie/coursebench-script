# CourseBench Script
将 Coursebench 的课程评价数据集成到上海科技大学选课系统中，便于快速查看课程评价并跳转。

# 食用方法

1. 浏览器安装 [油猴插件](https://www.tampermonkey.net/)
2. 点击 [这里](https://github.com/ShanghaitechGeekPie/coursebench-script/raw/refs/heads/main/coursebench-script.user.js) 安装脚本
3. 此时打开 EAMS 选课就能看到右侧多出的评价列了
